# COVID-19 Italian Situation Tracker

  Web app che mostra in tempo reale i dati COVID-19 italiani.

  ## Tech Stack
  - Python, Flask, Jinja2, HTML/CSS

  ## Come avviare
  ```bash
  pip install flask
  python application.py
  Apri il browser su http://127.0.0.1:5000  ```
